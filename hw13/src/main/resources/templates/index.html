books
Доступ с ролью user или admin.
Для пользователей с ролью User есть ACL на чтение и редактирование по жанру книгу и на саму книгу, если он ее создал
Для пользователей с ролью Admin доступны все книги и все операции над книгами
Доступные жанры пользователя указываются в пользователе
Если пользователь создал новую книгу, то книга ему будет доступна без ограничения по жанру
